# Rubicon

Rubicon is a shogi engine framework, written by Ruby.

Shogi is game like chess.

## Source Code Explanation

(1) atkop.rb : Functions of piece attacks.

(2) attacks.rb : Functions of attack tables.

(3) bitop.rb : Functions of bit operations.

(4) board.rb : Functions of shogi board.

(5) cap.rb : Functions for generating capture moves.

(6) check.rb : Functions for check moves.

(7) color.rb : Constants of color(or turn).

(8) common.rb : Common constants and variables.

(9) csa.rb : Functions of CSA format.

(10) direction.rb : Constants of directions.

(11) drop.rb : Functions for generating drop moves.

(12) evasion.rb : Functions for generating evasion moves.

(13) file.rb : Constants of file.

(14) hash256.rb : Functions of hash value.

(15) inout.rb : Functions for reading records.

(16) main.rb : The entry point of this software.

(17) makemove.rb : Functions to do move and undo move.

(18) move.rb : Functions for moves.

(19) nocap.rb : Functions for generating no capture moves.

(20) piece.rb : Constants of pieces.

(21) rank.rb : Constants of rank.

(22) record.rb : A class of record.

(23) sfen.rb : Functions for SFEN.

(24) test.rb : Functions for testing.

(25) usi.rb : Functions for USI.

* USI -> Universal Shogi Interface
